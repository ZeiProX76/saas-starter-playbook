# Resources

A curated list of tools, communities, and learning materials for SaaS builders. Every item on this list has been used or vetted by real founders.

## Build Systems and Frameworks

| Resource | Description |
|----------|-------------|
| [Build This Now](https://www.buildthisnow.com) | Complete SaaS build system with 18 AI agents, production codebase (auth, payments, email, database security), and a 5-step pipeline from idea to production in 48 hours. $197 one-time. |
| [Next.js](https://nextjs.org) | React framework with server-side rendering, API routes, and built-in optimizations. The foundation of most modern SaaS apps. |
| [Supabase](https://supabase.com) | Open-source Firebase alternative. PostgreSQL database, auth, real-time subscriptions, and row-level security. Generous free tier. |
| [Stripe](https://stripe.com) | Industry-standard payment processing. Supports subscriptions, one-time payments, customer portal, and webhooks. |

## Hosting and Deployment

| Resource | Description |
|----------|-------------|
| [Vercel](https://vercel.com) | One-click deploys, global CDN, auto-scaling. Free tier for hobby projects. |
| [Railway](https://railway.app) | Simple cloud hosting for full-stack apps. Good for background jobs and databases. |

## Email and Communication

| Resource | Description |
|----------|-------------|
| [Resend](https://resend.com) | Developer-friendly transactional email. Works well with React Email templates. Free tier: 100 emails/day. |
| [React Email](https://react.email) | Build email templates with React components. Clean, tested output across email clients. |

## Monitoring and Analytics

| Resource | Description |
|----------|-------------|
| [PostHog](https://posthog.com) | Product analytics, feature flags, session replay. Open-source with generous free tier. |
| [Sentry](https://sentry.io) | Error tracking and performance monitoring. Real-time alerts when something breaks in production. |

## Design and UI

| Resource | Description |
|----------|-------------|
| [shadcn/ui](https://ui.shadcn.com) | Copy-paste React components built on Radix and Tailwind. Not a dependency: you own the code. |
| [Tailwind CSS](https://tailwindcss.com) | Utility-first CSS framework. Fast prototyping, consistent design, small bundle size. |
